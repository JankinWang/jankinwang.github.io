# ES6 模块

### 导出

变量

```js
// 声明同时导出
export const username = '阿三';

// 先声明
const username = '阿三';
export username;

```



函数

```js
export function getName() {};

/* 或 */ 

function getName() {};
export getName;
```



类

```js
export class Student {};

/* 或 */

class Student {};
export Student;
```



一次性导出

```js
const username = '阿三';
function getName() {};
class Student {};
const obj = {};

export {
	username,
    getName as myName, // 别名
    Student,
    obj
}
```



默认导出

```js
// 一个模块只能有一个默认导出
export default class Student {};

// 作为别名提供的default关键字, 也会成为默认导出
const username = '阿三';
export {
	username as default
}
```



转移导出

```js
// 导出全部
export * from './module.js'

// 导出部分
export {getName as name, Student} from './module.js'

// 导出默认
export { default } from './module.js'
export { default * as name } from './module.js'

export { username as default } from './module.js'
```



### 导入

```js
// 导入默认(名字可与导出时不同)
import student from './module.js'
import { default as student } from './module.js'

// 导入指定（名字必须与导出时一样，可起别名）
import { Student as student, getName } from '.module.js'

// 导入全部（将所有导出集中导入到别名中）
import * as module from './module.js'

// 如果只想加载和执行模块
import './module.js'
```
