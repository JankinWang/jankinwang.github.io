# 运算符的扩展

  1. 指数运算符 (opens new window) 

    2 ** 2 // 4
2 ** 3 // 8

  2. 链判断运算符 (opens new window)

    const firstName = message?.body?.user?.firstName || 'default';

  3. [Null 判断运算符](https://es6.ruanyifeng.com/#docs/operator#Null 判断运算符)

    ?? 的行为类似||，但是只有运算符左侧的值为null或undefined时，才会返回右侧的值。

  4. 逻辑赋值运算符 (opens new window)

    // 或赋值运算符
x ||= y
// 等同于
x || (x = y)

// 与赋值运算符
x &&= y
// 等同于
x && (x = y)

// Null 赋值运算符
x ??= y
// 等同于
x ?? (x = y)
