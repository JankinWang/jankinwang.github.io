# 数组的扩展

  1. 扩展运算符 (opens new window)

  2. Array.from() (opens new window)

    将转类似数组的对象（array-like object）和可遍历（iterable）的对象（包括 ES6 新增的数据结构 Set 和 Map）

  3. Array.of() (opens new window)

    将一组值，转换为数组。

    Array.of(3, 11, 8) // [3,11,8]
Array.of(3) // [3]
Array.of(3).length // 1

  4. 实例方法：copyWithin() (opens new window)

    在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组。会修改当前数组。

  5. [实例方法：find() 和 findIndex()](https://es6.ruanyifeng.com/#docs/array#实例方法：find() 和 findIndex())

    • find方法，用于找出第一个符合条件的数组成员，它的参数是一个回调函数，所有数组成员依次执行该回调函数，直到找出第一个返回值为true的成员，然后返回该成员。如果没有符合条件的成员，则返回undefined。
    • findIndex方法的用法与find方法非常类似，返回第一个符合条件的数组成员的位置，如果所有成员都不符合条件，则返回-1。

  6. 实例方法：fill() (opens new window)

    使用给定值，填充一个数组。

    ['a', 'b', 'c'].fill(7)
// [7, 7, 7]

new Array(3).fill(7)
// [7, 7, 7]

  7. [实例方法：entries()，keys() 和 values()](https://es6.ruanyifeng.com/#docs/array#实例方法：entries()，keys() 和 values())

    ES6 提供三个新的方法——entries()，keys()和values()——用于遍历数组。

  8. 实例方法：includes() (opens new window)

    返回一个布尔值，表示某个数组是否包含给定的值，与字符串的includes方法类似。

  9. 实例方法：flat()，flatMap() (opens new window)

    将嵌套的数组“拉平”，变成一维的数组。该方法返回一个新数组，对原数据没有影响。
