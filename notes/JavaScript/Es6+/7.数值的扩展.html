# 数值的扩展

  1. 二进制和八进制表示法 (opens new window)
ES6 提供了二进制和八进制数值的新的写法，分别用前缀`0b`（或`0B`）和`0o`（或`0O`）表示。

  1. 数值分隔符 (opens new window)

    123_00 === 12_300 // true

12345_00 === 123_4500 // true
12345_00 === 1_234_500 // true

  2. [Number.isFinite(), Number.isNaN()](https://es6.ruanyifeng.com/#docs/number#Number.isFinite(), Number.isNaN())

  3. [Number.parseInt(), Number.parseFloat()](https://es6.ruanyifeng.com/#docs/number#Number.parseInt(), Number.parseFloat())

  4. Number.isInteger() (opens new window)

  5. Number.EPSILON (opens new window)

    Number.EPSILON实际上是 JavaScript 能够表示的最小精度

  6. [安全整数和 Number.isSafeInteger()](https://es6.ruanyifeng.com/#docs/number#安全整数和 Number.isSafeInteger())

  7. [Math 对象的扩展](https://es6.ruanyifeng.com/#docs/number#Math 对象的扩展)

    1. Math.trunc() (opens new window) 去除一个数的小数部分，返回整数部分

    2. Math.sign() (opens new window) 用来判断一个数到底是正数、负数、还是零。对于非数值，会先将其转换为数值。

    3. Math.cbrt() (opens new window) 计算一个数的立方根。

    4. Math.clz32() (opens new window) 将参数转为 32 位无符号整数的形式，然后返回这个 32 位值里面有多少个前导 0。

    5. Math.imul() (opens new window) 返回两个数以 32 位带符号整数形式相乘的结果，返回的也是一个 32 位的带符号整数。

    6. Math.fround() (opens new window) 返回一个数的32位单精度浮点数形式

    7. Math.hypot() (opens new window) 返回所有参数的平方和的平方根

    8. 对数方法 (opens new window)

    9. 双曲函数方法 (opens new window)

  8. [BigInt 数据类型](https://es6.ruanyifeng.com/#docs/number#BigInt 数据类型)

    const a = 2172141653n;
const b = 15346349309n;
