# 对象的新增方法

  1. Object.is() (opens new window)

    用来比较两个值是否严格相等，与严格比较运算符（===）的行为基本一致。

    不同之处只有两个：一是+0不等于-0，二是NaN等于自身。

  2. Object.assign() (opens new window)

    用于对象的合并，将源对象（source）的所有可枚举属性，复制到目标对象（target）

    Object.assign(target, source1, source2);

  3. Object.getOwnPropertyDescriptors() (opens new window)

    指定对象所有自身属性（非继承属性）的描述对象

  4. __proto__属性，Object.setPrototypeOf()，Object.getPrototypeOf() (opens new window)

    读取或设置当前对象的原型对象

  5. Object.keys()，Object.values()，Object.entries() (opens new window)

    功能依次是：返回对象自身的（不含继承的）所有可遍历（enumerable）属性的

    键名 键值 键值对 数组

  6. Object.fromEntries() (opens new window)

    Object.entries()的逆操作，用于将一个键值对数组转为对象。
