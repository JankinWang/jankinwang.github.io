# 伪类

### 元素状态

```css
:link /* 未访问时的链接 */
:visited /* 用户已访问过的链接 */

:hover
:focus /* 元素获得焦点 */
:focus-within /* 元素获得焦点，或，该元素的后代元素获得焦点。 */
:active


:disabled /* 禁用的 */
:enabled  /* 可用的，与disabled相反 */
:read-only /* 只读状态，不只是选择具有 readonly 属性的<input> 元素，它也会选择所有的不能被用户编辑的元素。 */
:read-write /* 可以被用户编辑的元素，例如设置了 contenteditable 属性的  <p> 元素。 */


/* 表单 */
:checked /* 选中的 radio checkbox 或("select") 元素中的option */
:default
:indeterminate /* 状态不确定的表单元素 */
:valid /* 通过验证的 <input> 或其他 <form> 元素 */
:invalid /* 任意内容未通过验证的 <input> 或其他 <form> 元素 */
:in-range /* 代表一个 <input> 元素，其当前值处于属性min 和max 限定的范围之内. */
:out-of-range /* 与 in-range 相反 */
:optional /* 可选字段，没有设置required的 <input>，<select> 或  <textarea>  */
:required /* 必填字段 */


:empty  /* 没有子元素的元素 子元素只可以是元素节点或文本（包括空格） */

```



### 选择器延伸

```css
:not()  /* :not(p) 选择所有不是段落（p）的元素 */

:first-child /* 父元素中的处于第一个位置的元素 */
:last-child

:first-of-type /* 兄弟元素中同一类型的第一个元素, 不管其在兄弟内的位置如何。 */
:last-of-type

:nth-child()
:nth-last-child()  /* 从最后开始计数 */

:nth-of-type()
:nth-last-of-type()

:only-child /* 父元素的唯一子元素，没有兄弟元素 */
:only-of-type /* 这个元素没有其他相同类型的兄弟元素 */
```



### 其他

```css
:target /* 其id 与当前URL片段匹配 如index.html#section2 */
:root /* 等同于 <html>, 优先级更高 */
:lang() /* (https://developer.mozilla.org/zh-CN/docs/Web/CSS/:lang) */

/* 打印 */
:first /* 打印文档的时候，第一页的样式 */
:left /* 对打印文档的左侧页设置CSS样式 */
:right

/* 没搞明白 */
:host
:host()
:scope
```

::selection
